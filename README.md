Student Performance Prediction
Overview

This project predicts student performance based on study hours using a machine learning model. It helps understand the relationship between study time and academic results.

Objective

To build a simple regression model that estimates student scores based on the number of hours studied.

Dataset Features
Study Hours
Student Scores

Tools Used
Python

Result

The model shows a clear positive relationship between study hours and student performance.
